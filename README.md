## Installation

```sh
git clone git@github.com:vian21/RocketForce.git && cd RocketForce
```

```sh
git submodule init && git submodule update
```

```sh
brew install sdl2
```

### Compile simulator

```sh
make
```

### Run simulator

```sh
make run
```
